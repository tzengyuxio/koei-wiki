# L'Empereur Person Editor - 拿破崙時代人物條目規範

本 Skill 提供自動化建立《拿破崙》（L-Empereur）遊戲人物條目的標準流程。

## 1. 核心檔案規範

- **檔名**：必須使用**遊戲中的中文名稱**（例如：`杜邦.md`、`拿破崙.md`）。
- **目錄**：預設存放於 `人物/拿破崙時代/`。
- **Frontmatter**：
    - `title`: 遊戲中文名。
    - `description`: 簡要描述人物身分或關係（參考 CSV 備註，排除技術錯誤備註）。
    - `tags`: 
        - 必須採用**單行表示式**（例如：`tags: [lempe, marshal, 法蘭西]`）。
        - 必須包含 `lempe`。
        - **嚴禁重複**標籤。

## 2. 列傳 (Bio) 撰寫標準

- **篇幅**：每篇必須介於 **300 至 1000 字**。嚴禁產生低質量的佔位符文字。
- **資料來源**：必須參考中/英文維基百科史實。若無資料，標註「（待考證）」。
- **命名規則**：正文提及人物時使用其真實世界名稱。
- **內連連結 (Inter-linking)**：
    - 當提及其他已存在（或潛在）的人物時，必須使用格式：`[真實姓名](/人物/拿破崙時代/遊戲中文名)`。
    - 範例：`他在[拉納](/人物/拿破崙時代/列寧斯)元帥麾下服役。`

## 3. 資源處理

- **肖像圖**：
    - 來源：優先從英文維基百科抓取 `og:image`。
    - 命名：轉換為小寫 snake_case（例如：`napoleon_bonaparte.jpg`）。
    - 存放：`assets/portraits/`。
- **自動化工具**：使用本機 `scripts/add_person.py` 進行實體檔案生成。

## 4. 工作流程

1. 從 `assets/data/lempereur/lempe-persons-full.csv` 檢索 ID 資料。
2. 呼叫 `web_fetch` 抓取生平與圖片連結。
3. 執行生成腳本並進行內容擴充。
